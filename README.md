# Decodelabs Task 1 - Data Cleaning Using Python

## Overview
This project focuses on cleaning and preprocessing an Excel dataset using Python and Pandas. The goal was to identify and handle missing values, detect duplicate records, and improve the overall quality of the data for further analysis.

## Objectives
- Identify missing values in the dataset
- Detect duplicate rows and records
- Handle incomplete data appropriately
- Improve data quality and consistency
- Prepare the dataset for further analysis and visualization

## Tools Used
- Microsoft Excel
- Python
- Visual Studio Code (VS Code)

## Tasks Performed
- Loaded the Excel dataset using Pandas
- Examined the dataset structure and data quality
- Checked for missing values across all columns
- Identified duplicate rows and duplicate Order IDs
- Filled missing CouponCode values with a default value
- Cleaned and organized the dataset for analysis

## Files Included
- Data Cleaning Python Script (.py)
- Excel Dataset (.xlsx)
- README.md

## Key Learning Outcomes
- Data loading and preprocessing using Pandas
- Handling missing values effectively
- Detecting and managing duplicate records
- Improving dataset quality for analytical purposes
- Applying fundamental data cleaning techniques used in real-world projects

## Outcome
The dataset was successfully cleaned by addressing missing values and duplicate records. The final dataset became more accurate, consistent, and ready for further analysis, reporting, and visualization.
